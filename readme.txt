Папки: 
Assets - хранятся папки с изображениями, стилями и скриптами JS //в данном случае только стили.

Файлы (по алфавиту, но не по значению):

404.php - скрипт, выводящий ошибку 404.
comment.php - скрипт, позволяющий оставлять комментарии.
db.php - скрипт, отвечает за подключения бд.
footer.php - верстка футера сайта, подключается через php в нужных местах. 
function.php - скрипт, подключает функцию для взаимодействия с бд.
header.php - верстка чердака сайта, подключается через php в нужных местах.
index.php - основная часть сайта. 
LK.php - личный кабинет, по сути корзина, также там модераторская для комментов.
single.php - страница конкретного товара с дополнительной информацией.
zd11.sql - база данных, для того, чтобы все работало, импортируй данный файл в свой phpMyAdmin.

Все взаимодействия с бд производятся через команды а-ля  $mysql = new mysqli('localhost','root','','kurs');
Если у тебя другие названия пользователя или есть пароль - конструкцию надо будет соответствующе изменить, например:
	$mysql = new mysqli('localhost','ПОЛЬЗОВАТЕЛЬ','ПАРОЛЬ','НАЗВАНИЕ БД');

Выводимая информация, пикчи товаров, их описание - в бд, наполнять контентом и редактировать через phpMyAdmin.
